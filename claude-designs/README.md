# Claude Designs

This folder contains all Markdown files that have been **converted into designed web pages** by Claude AI.

## What goes here?
- HTML files generated by Claude from the raw Markdown sources
- Each file should have a matching source file in `raw-markdown/`
- Styled, responsive, and ready-to-view design outputs

## Naming Convention
Keep the same filename as the source, just change the extension:
- Source: `raw-markdown/my-guide.md`
- Output: `claude-designs/my-guide.html`
