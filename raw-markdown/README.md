# Raw Markdown Files

This folder contains all the original raw Markdown (`.md`) files.

## What goes here?
- All source content written in plain Markdown format
- Notes, guides, documentation, design briefs
- Any file ending in `.md` before it is converted

## How to use
1. Add your `.md` files here
2. Once ready, send to Claude to convert into a designed HTML/web page
3. The converted output goes into the `claude-designs/` folder
