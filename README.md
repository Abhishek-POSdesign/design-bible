# design-bible

A structured repository for managing design documents — from raw Markdown drafts to Claude-powered designed outputs.

## Folder Structure

```
design-bible/
├── raw-markdown/      ← Original Markdown (.md) source files
└── claude-designs/    ← HTML/designed outputs converted by Claude AI
```

## Workflow

1. **Write** your content and save it as a `.md` file in `raw-markdown/`
2. **Convert** using Claude — paste the Markdown and ask Claude to design it
3. **Save** the HTML output into `claude-designs/` with the same filename

## Purpose
This repo acts as a design bible — a single source of truth for all design documents, guides, and references.
